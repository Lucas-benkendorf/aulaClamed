Dicionário de Tags HTML

Principais elementos HTML explicados.

<html>

Elemento raiz de uma página HTML. Envolve todo o conteúdo da página.

<head>

Contém metadados, links para estilos, scripts, e outras informações.

<title>

Define o título do documento.

<body>

Contém todo o conteúdo visível da página.

<h1> a <h6>

Títulos de diferentes níveis de importância, do mais ao menos importante.

<p>

Define um parágrafo.

<ul>

Define uma lista não ordenada.

<li>

Define um item em uma lista.

<strong>

Define texto importante em negrito.

<a>

Define um hiperlink.

<img>

Incorpora uma imagem. Atributo src especifica a fonte da imagem.
